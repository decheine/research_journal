## Notebooks

I really like notebooks as a way to display information.

**Strategy**

* Do Experiments
* Upload Notebook (meta data, sections, comments)
* Refactor Notebook (Clean, sections, labels, only important info)
* Re-Upload Notebook
* Organize and MKDocsify-it
